# PyProxy-Core 🛡️

Un serveur proxy HTTP/HTTPS minimaliste et performant écrit en Python. Idéal pour comprendre le fonctionnement des tunnels réseau et la manipulation de sockets.

## Fonctionnalités
- ✅ Support complet du protocole **HTTP**.
- ✅ Tunneling **HTTPS** (via méthode CONNECT).
- ✅ Multi-threading pour gérer plusieurs connexions simultanées.
- ✅ Facilement containerisable avec Docker.

## Installation Rapide

### Avec Python localement
```bash
python proxy.py
